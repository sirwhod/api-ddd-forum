# Subdomínios

- Core: O que dá dinheiro
- Supporting: Dá suporte para o core funcionar
- Generic: Você precisa, mas não são tão importântes

## Exemplos

Em um contexto de um e-commerce

### Core

- Compra
- Catálogo
- Pagamento
- Entrega
- Faturamento

### Supporting

- Estoque

### Generic

- Notificação
- Promoções
- Chat
